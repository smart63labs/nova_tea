Aba Totais

Após, deve abrir o DARE para emissão.

Dia: 27/11/2023

Nota Fiscal Tributada com Liminar

As notas fiscais de produtores que possuem mandado de segurança para não cobrança do ICMS, transferência interestadual entre propriedades de um mesmo produtor, devem ser emitidas tributadas. Para isso, após confirmar a NF, deve-se colocar código 422 e subcódigo 4.14 para que o DARE saia sem a cobrança do tributo.

Aba: Emissão de Notas Fiscais
