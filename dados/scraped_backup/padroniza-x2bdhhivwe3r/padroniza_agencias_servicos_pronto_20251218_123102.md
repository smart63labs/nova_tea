# Serviços PRONTO

Para usufruir dos serviços online da Secretaria da Fazenda do Tocantins, oferecidos através do portal PRONTO, o usuário PESSOA FÍSICA deve primeiramente realizar o cadastro no Domicílio Eletrônico de Contribuintes (DEC). Consulte o Manual DEC para obter orientações detalhadas sobre este processo. [Manual DEC](</view/padroniza-agencias/outros/dec-domicílio-eletrônico-do-contribuinte/manual-dec>). Este procedimento só é necessário fazer a primeira vez.

Após o cadastro no DEC, será necessário efetuar também o registro no Portal de Serviços. Para auxiliar nesta etapa, disponibilizamos o Manual de Cadastro. [Manual Cadastro Pronto](</view/padroniza-agencias/outros/servicos-pronto/manual-cadastro-pronto>). Este procedimento só é necessário fazer a primeira vez.

Assunção de Dívida de IPVA

Para obter informações sobre como solicitar a Assunção de Dívida de IPVA, consulte nosso manual detalhado. [Clique aqui](</view/padroniza-agencias/outros/servicos-pronto/manual-assunção-do-ipva>) para acessá-lo. 

Solicitação de Cancelamento de Notas Fiscais Avulsas Eletrônicas - NFA-e

Para obter informações sobre como solicitar o cancelamento de NFA-e, consulte nosso manual detalhado. [Clique aqui](</view/padroniza-agencias/outros/servicos-pronto/cancelamento-nf>) para acessá-lo.
